## ✨ Fitur

### 🎯 Teknologi Utama
| Fitur | Keterangan |
|-------|-----------|
| **HTML5** | Markup semantik dengan template includes |
| **Tailwind CSS v3** | CSS utility-first dengan JIT compilation |
| **SCSS/Sass** | Syntax modern `@use` untuk organisasi |
| **Alpine.js** | Framework JavaScript yang ringan |
| **Gulp v4** | Otomasi task & build pipeline |
| **PostCSS** | Transformasi CSS & autoprefixer |

### 🚀 Fitur Pengembangan
- ⚡ **Live Reload** - Refresh browser otomatis
- 🎨 **Hot CSS** - Update style instant
- 📱 **Mobile First** - Desain responsif dari awal
- 🌙 **Dark Mode** - Penggantian tema berbasis class
- 🎭 **Animasi** - Transisi & efek smooth
- 📦 **Build Produksi** - Output yang dioptimalkan & diminify

### 🎨 Sistem Desain
- 🎯 Custom breakpoint (xs, sm, tablet, lg, desktop, xl, 2xl)
- 🎨 CSS Variables untuk theming dinamis
- 📚 Dua font custom (Lato & SUSE)
- 🎭 Arsitektur komponen yang dapat digunakan kembali
- ✨ Sistem shadow & border-radius
- 🎬 Animasi bawaan

### 🛠️ Pengalaman Developer
- 📁 Struktur file yang terorganisir dengan baik
- 🔍 Pesan error yang jelas
- 📝 Komponen SCSS modular
- 🧩 Perpustakaan komponen reusable
- 💾 Task cleanup otomatis
- ✅ Validasi build

## 🚀 Mulai Cepat

### Prasyarat
```bash
# Diperlukan: Node.js v24+ dan npm v10+
node --version   # v24.12.0+
npm --version    # v10.0.0+

Instalasi (3 langkah)

# 1️⃣ Clone repository
git clone https://github.com/farreldev/tailwindcss-boilerplate.git
cd tailwindcss-boilerplate

# 2️⃣ Install dependencies
npm install

# 3️⃣ Jalankan server development
npm run dev

🎉 Browser membuka di http://localhost:9010

Development

npm run dev

🔄 Live reload dengan auto-refresh
👀 Watch file untuk perubahan
🌐 Buka browser di http://localhost:9010
💾 Compile SCSS ke CSS
🎨 Proses Tailwind utilities
