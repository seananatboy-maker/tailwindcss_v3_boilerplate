Fitur Unggulan

Semua yang Anda Butuhkan, Dalam Satu Paket

Dibangun dengan best practices dan tools modern agar Anda bisa langsung fokus membuat website tanpa setup yang rumit.

Tailwind CSS v3

Utility-first CSS framework dengan JIT compilation untuk styling cepat dan efisien.

Gulp v4 Automation

Task runner otomatis untuk compile, watch, live reload, dan optimasi build produksi.

SCSS / Sass

Arsitektur SCSS modular dengan syntax @use modern untuk organisasi style yang rapi.

Live Reload

Auto-refresh browser setiap kali ada perubahan file. Proses develop jadi super cepat.

Gulp File Include

Sistem template include untuk memecah HTML menjadi komponen reusable yang modular.

Production Ready

Output yang dioptimalkan, diminify, dan siap deploy ke production environment.

Quick Start

Mulai dalam 3 Langkah

Cukup clone, install, dan jalankan. Tidak perlu konfigurasi rumit.

1

Clone Repository

git clone https://github.com/seananatboy-maker/tailwindcss_v3_boilerplate.git
2

Install Dependencies

cd tailwindcss_v3_boilerplate
npm install
3

Jalankan Development Server

npm start
Browser terbuka di http://localhost:9010
3

Jalankan Build Production

npm run build
Membuat folder "./dist" diroot direktori dengan file siap deploy di dalamnya

Atau langsung copy URL clone berikut:

$
